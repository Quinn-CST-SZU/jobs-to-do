### 文件编码转化

iconv

`iconv [选项...] [文件...]`

**选项**

-f  --from-code=名称 原始文本编码
-t  --to-code=名称 输出编码

-l  --list 列举所有已知的字符集

-c 从输出中忽略无效的字符
-o, --output=FILE 输出文件
-s, --silent 关闭警告
--verbose 打印进度信息

**Example**

```shell
# gbk转utf8
iconv -f gbk -t utf-8 -c  text.txt -o text.out
# 把text.txt中的文件<content></content>间的文本取出，并且把编码从gbk转化为utf-8保存
cat text.txt | iconv -f gbk -t utf-8 -c | grep "<content>"  -o text.out
```
